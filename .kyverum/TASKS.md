## Prioridades
- **P0** = Crítico, se hace primero (bloquea todo lo demás)
- **P1** = Importante, se hace pronto
- **P2** = Deseable, se hace cuando haya tiempo
- **P3** = Nice-to-have, se hace si sobra tiempo

## Estados
- **pendiente** = No se ha empezado
- **en-progreso** = Alguien está trabajando en esto
- **en-revision** = Terminado, esperando QA
- **terminada** = Completada y verificada
- **bloqueada** = No se puede avanzar (hay un problema)
